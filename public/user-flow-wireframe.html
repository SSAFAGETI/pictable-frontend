User Journey Flow Chart

같은 사용자 여정을 기준으로 앱/웹 화면만 토글해 확인합니다. 노드를 누르면 해당 화면 카드로 이동합니다.

실선: 주요 이동 분기: 권한/상태 앱 전용: 요리모드
메인 여정 처음 진입부터 레시피 상세까지 이어지는 핵심 흐름
01AUTH
Login / Signup
이메일 또는 구글로 가입하고 환영 알림을 생성합니다.
/login · /signup
→
03HOME
Home Discovery
추천 요리, 인기 레시피, 최근 마이 레시피를 탐색합니다.
/
→
04FEED
Search / Tag Filter
검색어와 태그 AND 필터로 원하는 레시피를 좁힙니다.
/feed
→
06DETAIL
Recipe Detail
좋아요, 저장, 댓글, 작성자 수정 권한이 모이는 중심 화면입니다.
/recipe/{id}
미로그인 액션 → 로그인 유도
로그인 사용자 → 소셜 액션 가능
작성자 → 수정/답글 가능
↓
상세 액션 레시피 상세에서 발생하는 저장, 댓글, 알림 흐름
07COMMENT
Comment Thread
댓글과 작성자 답글을 한 상세 페이지 안에서 이어 봅니다.
detail comments
→
13NOTICE
Notifications
환영, 좋아요, 댓글 알림을 카운트 배지와 드롭다운으로 확인합니다.
Header Notification
→
10SAVED
Saved Recipes
저장한 레시피를 다시 찾아 상세 페이지로 복귀합니다.
/saved
개인화 흐름 내 레시피 등록, 수정, 추천으로 확장되는 흐름
11MY
My Page
내 활동과 저장한 레시피, API 명세 진입점을 관리합니다.
/mypage
→
08CREATE
Register Recipe
새 레시피를 등록하고 최근 마이 레시피에 반영합니다.
/my-recipe/new
→
09EDIT
Edit My Recipe
내가 만든 레시피는 상세 페이지에서 수정 흐름으로 연결됩니다.
/my-recipe/new?edit
추천과 요리모드 재료 기반 추천과 모바일 가로모드 조리 UX
12RECOMMEND
Ingredient Recommendation
냉장고 재료 입력 또는 인식 결과로 추천 레시피를 확인합니다.
/recommendations
→
14APP ONLY
Cooking Mode
모바일 가로 화면에서 단계별 조리와 완료 팝업을 진행합니다.
landscape mode
→
ENDLOOP
Like / Save / Return
요리 완료 후 좋아요, 저장, 상세 복귀로 다시 탐색 흐름에 합류합니다.
back to detail
